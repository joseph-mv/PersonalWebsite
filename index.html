TripTide

A responsive and feature-rich travel planning website built using the MERN stack, designed to suggest locations along routes and enable users to create and save personalized itineraries.

Features:
  • Location Suggestions & Itinerary Creation: Automatically generates forms based on selected travel days, allowing users to create and save detailed itineraries with suggested locations along their routes.
  • User Authentication: Includes secure signup, email verification, login, and password recovery using JWT tokens with refresh tokens, and bcrypt for password protection.
  • Location Database Integration: Utilizes WikiQuery to create a location database with coordinates, integrating data from Mapbox, Unsplash, Wikidata, and Wikipedia.
  • Interactive Maps: Allows users to view and interact with maps, visualizing routes and selected locations for a comprehensive travel experience.
  • Security & Email Verification: Uses NodeMailer for email verification and OTP, ensuring account security.
  • State Management & Animations: Implements Redux for efficient state management and Framer Motion for smooth animations, with LazyLoadImage optimizing image loading.
  • Scalability: Backend hosted on AWS EC2, ensuring reliable performance and scalability.
Tech Stack:

  • Redux: State management library for React applications.
  • Framer Motion: Animation library for smooth transitions and effects.

  • AWS EC2: Cloud hosting service for backend scalability.
